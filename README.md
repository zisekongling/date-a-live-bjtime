# 北京时间网站

一个显示北京时间的网站，时间数字使用约会大作战人物图片显示。

## 项目结构
```
beijing-time/
├── index.html          # 主页面
├── script.js           # JavaScript代码
├── README.md           # 说明文档
└── resources/          # 资源文件夹
    ├── 0/              # 数字0的图片
    ├── 1/              # 数字1的图片
    ├── 2/              # 数字2的图片
    ├── 3/              # 数字3的图片
    ├── 4/              # 数字4的图片
    ├── 5/              # 数字5的图片
    ├── 6/              # 数字6的图片
    ├── 7/              # 数字7的图片
    ├── 8/              # 数字8的图片
    ├── 9/              # 数字9的图片
    ├── 10/             # 数字10的图片（用于倒计时天数）
    ├── 十/             # 中文“十”的图片
    └── 万/             # 中文“万”的图片
```

## 使用方法

1. 在`resources`文件夹下的对应数字文件夹中添加图片文件
   - 例如：在`resources/0/`文件夹中添加`0-1.jpg`、`0-2.jpg`等图片
   - 一个数字可以有多个图片，系统会随机选择显示
   - 相同数字在秒数变化时如果保持不变，图片也不会改变

2. 用浏览器打开`index.html`文件即可查看效果

## 页面布局

- 显示格式：xxxx年xx月xx日
- 时间格式：XX:XX
- 祝福语：十全十美，万事如意（“十”和“万”使用图片显示）
- 倒计时：距离明年还剩XXX天

## 注意事项

- 图片文件命名格式：`数字-序号.jpg`（例如：`0-1.jpg`、`5-3.jpg`）
- 支持的图片格式：jpg、png等浏览器支持的格式
- 图片尺寸建议：高度统一，宽度自适应
- 确保图片文件路径正确，否则会显示文字替代

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge等）。
